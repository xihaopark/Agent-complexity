# Agent-complexity

- **`front/`** — Agentic complexity analyzer application (Python analyzers, API, workers, Next.js UI). Project metadata: `front/pyproject.toml`.
- **`main/finish/`** — Snakemake `-finish` workflows, benchmark reports (`ALL_EVAL.md`, `FINISH_EXPANSION_STATUS.md`, …), tools, and `workflow_candidates/` source trees.

Large or reproducible assets (reference genomes, conda env caches, `.snakemake/`, local benchmark run outputs) are listed in `.gitignore`. Restore inputs using each workflow’s `know-how.md` / upstream README.
