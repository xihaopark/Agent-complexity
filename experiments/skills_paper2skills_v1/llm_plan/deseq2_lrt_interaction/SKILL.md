## Implementation Approach

Standard approach for interaction testing:

```r
library(DESeq2)

# Load
counts <- read.table("input/counts.tsv", header=TRUE, row.names="gene_id")
coldata <- read.table("input/coldata.tsv", header=TRUE, row.names="sample")

# Full model with interaction
dds <- DESeqDataSetFromMatrix(
  countData=counts,
  colData=coldata,
  design = ~ genotype + treatment + genotype:treatment
)

# Filter and run
dds <- dds[rowSums(counts(dds)) >= 10, ]
dds <- DESeq(dds)

# Method 1: Wald test for specific interaction coefficient
res <- results(dds, name="genotypeKO.treatmenttreated")

# Method 2: LRT (alternative)
# dds <- DESeq(dds, test="LRT", reduced=~ genotype + treatment)
# res <- results(dds)

# Output
write.csv(as.data.frame(res), "output/interaction_de.csv")
```
