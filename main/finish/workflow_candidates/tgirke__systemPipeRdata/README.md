_systemPipeRdata_ is part of the _systemPipeR_ workflow management system
(WMS). It allows users to load preconfigured data analysis workflow templates
onto their system. Once loaded, the WMS can immediately utilize these templates
to conduct end-to-end analysis for diverse data analysis needs.

For additional information, please visit the project site [here](https://systempipe.org/about/project/#core-packages).
