- **Input**: `input/featureCounts_output.txt` with columns: Geneid, Chr, Start, End, Strand, Length, sample_0..sample_5.
- **Preprocess**: Remove rows starting with `ERCC-`.
- **DESeq2 Analysis**:
  - Extract count data (columns 6-11) and create condition factor (A for first 3 samples, B for last 3).
  - Create DESeq2 dataset and run DESeq2.
  - Filter results to remove rows with NA in log2FoldChange or padj.
- **Output**:
  - `output/deseq2_up.txt`: Genes with log2FoldChange >= 2.
  - `output/deseq2_down.txt`: Genes with log2FoldChange <= -2.
- **Validation**: Ensure outputs are written with `write.table(..., col.names=TRUE, row.names=TRUE, quote=FALSE)`. Check presence of `output/deseq2_up.txt`.
