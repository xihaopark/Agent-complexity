2. Extract the sample name from each file's basename (e.g., `sampleA` from `sampleA.counts.txt`).
3. Load each file into R, keeping only the `Geneid` and sample count columns.
4. Perform an outer join on the `Geneid` column across all three datasets.
5. Drop the `Geneid` column from the final merged dataset and set it as rownames.
6. Write the merged data to `output/merged_counts.tsv` with `quote=FALSE`, `col.names=NA`, and tab-delimited.
