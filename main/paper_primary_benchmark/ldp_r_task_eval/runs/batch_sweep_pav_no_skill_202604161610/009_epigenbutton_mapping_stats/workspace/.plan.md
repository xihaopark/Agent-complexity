## Plan

- **Input**: Read `input/mapping_stats.tsv` with columns `Line, Tissue, Sample, Rep, Reference_genome, Total_reads, Passing_filtering, All_mapped_reads, Uniquely_mapped_reads`.
- **Transformations**:
  - Extract numeric values from columns `Passing_filtering`, `All_mapped_reads`, and `Uniquely_mapped_reads`.
  - Calculate `Multi-mapping` as `All_mapped_reads - Uniquely_mapped_reads`.
  - Calculate `Unmapped` as `Total_reads - All_mapped_reads`.
  - Calculate `Filtered` as `Total_reads - Passing_filtering`.
- **Visualization**:
  - Create a two-panel PDF with stacked fill/stack bars per `(Line/Tissue/Sample/Rep)`.
  - Facet by `Line`.
  - Title: `Mapping statistics for ChIP_demo samples`.
  - Dimensions: Landscape, 10x12 inches.
- **Output**: Save the plot as `output/mapping_stats.pdf`.
- **Validation**: Ensure `output/mapping_stats.pdf` is created and visually check the plot for correctness.
