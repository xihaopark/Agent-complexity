1. **Input Data**: Read `input/mapping_stats.tsv` which contains columns: Line, Tissue, Sample, Rep, Reference_genome, Total_reads, Passing_filtering, All_mapped_reads, Uniquely_mapped_reads.
2. **Data Transformation**:
   - Extract numeric values from columns formatted as `<count> (<pct>%)`.
   - Calculate Multi-mapping reads as `All_mapped_reads - Uniquely_mapped_reads`.
   - Calculate Unmapped reads as `Total_reads - All_mapped_reads`.
   - Calculate Filtered reads as `Total_reads - Passing_filtering`.
3. **Visualization**:
   - Create a two-panel PDF with stacked bar plots.
   - Each bar represents breakdown into Uniquely mapped, Multi-mapping, Unmapped, and Filtered reads.
   - Facet the plots by `Line`.
   - Title the plot `Mapping statistics for ChIP_demo samples`.
   - Set PDF dimensions to 10x12 inches, landscape.
4. **Output**: Save the PDF as `output/mapping_stats.pdf`.
5. **Validation**: Ensure the PDF is correctly formatted and saved in the output directory.
