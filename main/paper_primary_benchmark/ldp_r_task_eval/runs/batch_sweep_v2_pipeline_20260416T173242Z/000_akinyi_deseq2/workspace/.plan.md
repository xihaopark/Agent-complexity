1. Read the count matrix from 'input/featureCounts_output.txt'.
2. Load the DESeq2 library in R.
3. Prepare the count matrix by removing rows starting with 'ERCC-'.
4. Set up the DESeq2 dataset using the count matrix and condition information.
5. Run the DESeq2 analysis to obtain differential expression results.
6. Filter the results to remove rows with NA values in log2FoldChange or padj.
7. Split the results into 'up' and 'down' based on log2FoldChange thresholds.
8. Write the 'up' results to 'output/deseq2_up.txt'.
9. Write the 'down' results to 'output/deseq2_down.txt'.
10. Verify the existence of 'output/deseq2_up.txt' and call submit_done(success=true).
