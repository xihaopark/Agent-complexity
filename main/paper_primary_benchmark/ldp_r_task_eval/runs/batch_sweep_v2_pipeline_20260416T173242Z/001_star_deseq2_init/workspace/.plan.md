1. Read the input files `counts.tsv` and `samples.tsv` to obtain the count matrix and sample information.
2. Filter out rows from the count matrix where the total count across all samples is less than or equal to 1.
3. Load the DESeq2 library in R and create a DESeqDataSet object using the filtered count matrix and sample information, with the design formula `~condition`.
4. Relevel the `condition` factor to set `untreated` as the base level.
5. Run the DESeq function on the DESeqDataSet object to perform differential expression analysis.
6. Save the DESeqDataSet object to `output/dds.rds`.
7. Extract normalized counts from the DESeqDataSet object and write them to `output/normalized_counts.tsv`.
8. Verify the existence of `output/normalized_counts.tsv` and call `submit_done(success=true)` if it exists.
