1. Load the DESeq2 library in R.
2. Read the DESeq2 object from 'input/dds.rds'.
3. Perform differential expression analysis using the contrast 'condition treated vs untreated'.
4. Apply the lfcShrink function with type='ashr' to obtain shrunken log2 fold changes.
5. Sort the results by adjusted p-value (padj).
6. Prepare the results data frame with columns: gene, baseMean, log2FoldChange, lfcSE, pvalue, padj.
7. Write the results to 'output/contrast_results.tsv' as a tab-delimited file without row names and quotes.
8. Verify the output file exists and call submit_done(success=true) if it does.
