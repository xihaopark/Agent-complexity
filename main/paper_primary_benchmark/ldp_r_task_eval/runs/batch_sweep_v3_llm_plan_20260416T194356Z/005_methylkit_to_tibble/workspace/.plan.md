1. Load the necessary R libraries: methylKit, dplyr, tidyr, and readr.
2. Read the methylKit object from the RDS file located at `input/mk_united.rds`.
3. Convert the methylKit object to a data frame and use `pivot_longer` to transform it into a long tibble format with columns `(chr, start, end, strand, metric, value, sample)`.
4. Save the long tibble to `output/df_mku.rds`.
5. Pivot the long tibble to a wide format on the `metric` column.
6. Calculate the mCpG value as the ratio of `numCs` to `coverage`.
7. Group the data by `sample` and `chr` and compute the mean mCpG for each group.
8. Write the resulting table to `output/mean_mcpg.tsv` using `readr::write_tsv`.
9. Verify the existence of `output/mean_mcpg.tsv` and call `submit_done(success=true)` if it exists.
