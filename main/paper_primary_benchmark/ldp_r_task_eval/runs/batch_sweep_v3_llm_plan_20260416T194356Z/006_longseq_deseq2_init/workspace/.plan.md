1. Load the DESeq2 library in R.
2. Read the counts matrix from 'input/all_counts.tsv'.
3. Read the samples metadata from 'input/samples.tsv'.
4. Create a DESeqDataSet object using the counts matrix and samples metadata with the design formula '~condition'.
5. Filter out rows from the DESeqDataSet where the total count is 10 or less.
6. Run the DESeq function on the filtered DESeqDataSet.
7. Save the DESeq2 object to 'output/dds.rds'.
8. Extract normalized counts from the DESeq2 object.
9. Write the normalized counts to 'output/normalized_counts.tsv' with 'Reference' as the first column and samples as subsequent columns.
10. Verify the existence of 'output/normalized_counts.tsv' and call submit_done(success=true) if present.
