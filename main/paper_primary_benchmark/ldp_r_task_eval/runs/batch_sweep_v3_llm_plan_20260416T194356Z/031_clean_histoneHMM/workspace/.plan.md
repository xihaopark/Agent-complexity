1. Load the necessary R package `rtracklayer` for handling GFF and BED files.
2. Define a function `process_sample` to:
   - Import a GFF file using `import.gff`.
   - Filter the data to retain only features with `avg_posterior >= 0.5`.
   - Export the filtered data to a GFF3 file using `export.gff3`.
   - Set the `score` attribute to `avg_posterior` and export to a BED file using `export.bed`.
3. Apply the `process_sample` function to both `sampleA` and `sampleB` GFF files, saving the outputs in the `output/` directory.
4. Verify the existence of the output files, especially `output/sampleA_avgp0.5.bed`.
5. If all outputs are correctly generated, call `submit_done(success=true)`.
