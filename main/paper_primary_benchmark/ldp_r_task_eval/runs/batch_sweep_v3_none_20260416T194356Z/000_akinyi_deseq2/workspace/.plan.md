1. Read the featureCounts-style count matrix from `input/featureCounts_output.txt`.
2. Filter out rows starting with `ERCC-`.
3. Prepare the DESeq2 dataset using the filtered count matrix.
4. Run DESeq2 differential expression analysis with `design = ~condition`.
5. Filter the results to remove rows where log2FoldChange or padj is NA.
6. Split the results into upregulated genes (log2FoldChange >= 2) and downregulated genes (log2FoldChange <= -2).
7. Write the upregulated genes to `output/deseq2_up.txt` and downregulated genes to `output/deseq2_down.txt` using `write.table` with specified parameters.
8. Verify the existence of `output/deseq2_up.txt` and call `submit_done(success=true)` if present.
