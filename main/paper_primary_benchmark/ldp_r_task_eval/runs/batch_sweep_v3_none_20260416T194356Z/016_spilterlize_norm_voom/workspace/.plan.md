1. Read the `input/filtered_counts.csv` file to load the gene expression data.
2. Use the `edgeR::DGEList` function to create a DGEList object from the data.
3. Apply `calcNormFactors` with the method set to 'TMM' to normalize the data.
4. Use `limma::voom` with `normalize.method='none'`, `span=0.5`, and `plot=TRUE` to transform the data.
5. Extract the `E` matrix from the `voom` results.
6. Write the `E` matrix to `output/normalized_counts.csv` using `data.table::fwrite` with `row.names=TRUE`.
7. Verify the output file exists and call `submit_done(success=true)`.
