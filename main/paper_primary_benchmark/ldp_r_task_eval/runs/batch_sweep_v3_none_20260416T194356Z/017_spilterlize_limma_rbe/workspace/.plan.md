1. Read the input files `input/normalized.csv` and `input/annotation.csv`.
2. Load necessary R libraries: `limma` for batch effect removal and `data.table` for writing the output.
3. Convert the data from `normalized.csv` to a matrix and the annotation data from `annotation.csv` to a data frame.
4. Use `limma::removeBatchEffect` to adjust for batch effects in the data using the batch information from the annotation file.
5. Write the adjusted data to `output/integrated_data.csv` using `data.table::fwrite` with row names included.
6. Verify the output file exists and call `submit_done(success=true)` if successful.
