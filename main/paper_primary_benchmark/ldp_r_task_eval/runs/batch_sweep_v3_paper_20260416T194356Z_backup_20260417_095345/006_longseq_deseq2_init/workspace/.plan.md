1. Read the input files `all_counts.tsv` and `samples.tsv` to obtain the counts matrix and sample information.
2. Load the DESeq2 library in R.
3. Create a DESeqDataSet object using the counts matrix and sample information with the design formula `~condition`.
4. Filter out rows where the total count is less than or equal to 10.
5. Run the DESeq function on the DESeqDataSet object.
6. Extract normalized counts from the DESeqDataSet object.
7. Save the DESeqDataSet object to `output/dds.rds`.
8. Write the normalized counts to `output/normalized_counts.tsv`.
9. Verify the existence of `output/normalized_counts.tsv` and call `submit_done(success=true)` if it exists.
