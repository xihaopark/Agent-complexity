1. Read the input count matrix from `input/counts.tsv` and the sample information from `input/samples.tsv`.
2. Filter out genes with total counts less than or equal to 1 from the count matrix.
3. Load the DESeq2 library in R and prepare the DESeqDataSet object using the filtered count matrix and sample information.
4. Relevel the condition factor to set 'untreated' as the base level.
5. Run the DESeq function to perform the differential expression analysis.
6. Extract the normalized counts from the DESeqDataSet object.
7. Save the DESeqDataSet object to `output/dds.rds`.
8. Write the normalized counts to `output/normalized_counts.tsv`.
9. Verify the existence of `output/normalized_counts.tsv` and call `submit_done(success=true)` if it exists.
