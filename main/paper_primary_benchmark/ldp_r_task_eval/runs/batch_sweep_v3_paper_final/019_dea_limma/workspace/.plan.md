1. Read the input files `counts.tsv` and `metadata.tsv` to obtain the gene expression data and sample metadata.
2. Load the necessary R libraries, including `limma` and `edgeR`.
3. Normalize the counts data using TMM normalization.
4. Use `voom` to convert the normalized counts to log-counts per million with associated precision weights.
5. Create a design matrix based on the `treatment` column from the metadata.
6. Fit a linear model using `lmFit` and apply empirical Bayes moderation using `eBayes`.
7. Extract the top table results and save them to `output/dea_results.csv`.
8. Save the fitted `lmFit` object to `output/lmfit.rds`.
9. Save the design matrix to `output/model_matrix.csv`.
10. Verify the existence of `output/dea_results.csv` and call `submit_done(success=true)` if all deliverables are present.
