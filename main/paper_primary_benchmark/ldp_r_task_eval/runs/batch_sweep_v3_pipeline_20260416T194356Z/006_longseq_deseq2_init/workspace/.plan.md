1. Read the input files `input/all_counts.tsv` and `input/samples.tsv`.
2. Load the DESeq2 library in R.
3. Create a DESeqDataSet object using the counts matrix and sample information with the design formula `~condition`.
4. Filter out rows where the total count is less than or equal to 10.
5. Run the DESeq function to perform differential expression analysis.
6. Save the DESeqDataSet object to `output/dds.rds`.
7. Extract normalized counts and write them to `output/normalized_counts.tsv`.
8. Verify the existence of `output/normalized_counts.tsv` and call `submit_done(success=true)` if it exists.
