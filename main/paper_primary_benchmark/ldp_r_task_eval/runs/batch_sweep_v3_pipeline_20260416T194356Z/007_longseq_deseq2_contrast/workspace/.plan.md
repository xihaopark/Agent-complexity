2. Read the DESeq2 object from 'input/dds.rds'.
3. Perform differential expression analysis using the `results` function with the contrast `c('condition','ko','wt')` and `alpha=0.05`.
4. Apply log fold change shrinkage using `lfcShrink` with `type='ashr'.
5. Sort the results by adjusted p-value (padj).
6. Write the sorted results to 'output/contrast_results.tsv' with the specified columns.
