1. Load the necessary R libraries for differential expression analysis using limma.
2. Read the expression data from `input/exprs.csv` and the metadata from `input/meta.csv`.
3. Prepare the design matrix using the `group` column from the metadata.
4. Fit the linear model using `lmFit` from the limma package.
5. Define the contrast for differential expression analysis using `makeContrasts`.
6. Apply the contrast to the fitted model and compute statistics using `eBayes`.
7. Extract the top 250 differentially expressed probes using `topTable` with FDR adjustment.
8. Save the results to `output/deg_results.csv`.
9. Verify the output file exists and call `submit_done(success=true)` if successful.
