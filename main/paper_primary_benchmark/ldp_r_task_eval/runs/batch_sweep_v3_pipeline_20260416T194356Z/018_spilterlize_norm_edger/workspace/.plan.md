1. Read the input CSV file using data.table::fread without row.names argument.
2. Create a DGEList object using edgeR::DGEList.
3. Normalize the data using edgeR::calcNormFactors with method='TMM'.
4. Calculate log-CPM using edgeR::cpm with log=TRUE and prior.count=3.
5. Write the log-CPM matrix to 'output/all/normTMM.csv' using data.table::fwrite with row.names=TRUE.
