1. Read the input files: `genecount.tsv`, `targets.tsv`, and `ref_genes.bed`.
2. Filter out rows from `genecount.tsv` where `GID` matches the pattern `^N_`.
3. Parse the `GID` from `ref_genes.bed` and prepare it for joining.
4. For each unique `Sample` in `targets.tsv`, identify the corresponding columns in `genecount.tsv`.
5. Compute the average gene count across replicates for each `Sample`.
6. Join the averaged counts with the reference gene table using the parsed `GID`.
7. Calculate the RPKM for each gene using the formula: `RPKM = avg * 1000 / (Stop - Start)`.
8. Accumulate the results into a single table `all_rpkm` with columns `GID, Sample, RPKM`.
9. Write the `all_rpkm` table to `output/results/RNA/DEG/genes_rpkm__runX__mockref.txt`.
10. Verify the output file exists and call `submit_done(success=true)`.
