# Examples

- Default bundled example:
  - `slurm/envs/deepchem/bin/python skills/drug-discovery-and-cheminformatics/deepchem-circular-featurization/scripts/compute_circular_fingerprints.py`
- Saved-output example:
  - `slurm/envs/deepchem/bin/python skills/drug-discovery-and-cheminformatics/deepchem-circular-featurization/scripts/compute_circular_fingerprints.py --out skills/drug-discovery-and-cheminformatics/deepchem-circular-featurization/assets/aspirin_caffeine_fingerprints.json`
- Single-molecule example:
  - `slurm/envs/deepchem/bin/python skills/drug-discovery-and-cheminformatics/deepchem-circular-featurization/scripts/compute_circular_fingerprints.py --smiles CCO --size 16 --radius 2`
