# Examples

The starter generates its own deterministic toy image, so no external input file is required.

Run the local example with:

```bash
./slurm/envs/scientific-python/bin/python \
  skills/imaging-and-phenotype-analysis/skimage-regionprops-feature-extraction/scripts/run_skimage_regionprops_features.py \
  --threshold 0.35 \
  --out scratch/skimage-regionprops/example_summary.json
```
