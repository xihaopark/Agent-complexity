# Verification Assets

- `hello_run_summary.json` records a successful local run through the repo-managed Nextflow environment.
- `hello_slurm_summary.json` records a successful Slurm-backed run with trace and accounting details.
- `hello_run_outputs/` preserves the first verified set of greeting files.
- `latest-results/` may be regenerated by future smoke runs.
- `latest-slurm-results/` may be regenerated by future Slurm smoke runs.
