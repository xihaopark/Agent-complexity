# Example

This folder contains a minimal `Snakefile`, one toy text input, and a two-step toy workflow.

Run it through the repo-managed prefix and a scratch workspace:

```bash
python3 skills/reproducible-workflows/snakemake-toy-workflow-starter/scripts/run_snakemake_workflow.py \
  --workspace scratch/snakemake-toy-workflow \
  --summary-out scratch/snakemake-toy-workflow/run_summary.json
```
