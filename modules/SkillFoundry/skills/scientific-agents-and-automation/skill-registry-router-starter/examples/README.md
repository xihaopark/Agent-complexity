# Example Queries

The router is query-driven and uses the live registry.

Examples:

```bash
python3 skills/scientific-agents-and-automation/skill-registry-router-starter/scripts/route_skill_query.py \
  --query "Bayesian regression with uncertainty intervals" \
  --top-k 3

python3 skills/scientific-agents-and-automation/skill-registry-router-starter/scripts/route_skill_query.py \
  --query "vector spatial join for points and polygons" \
  --top-k 3
```
