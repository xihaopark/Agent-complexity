# Inverse problems and data assimilation Starter

Use this starter when a task lands in the `Inverse problems and data assimilation` frontier leaf and the repository has curated resources but no dedicated runtime implementation yet.

## What this starter does

- Summarizes the local resource anchors for the leaf.
- Emits a machine-readable starter plan with promotion steps.
- Gives the agent a stable local entry point before a full runtime skill exists.

## How to use it

Run `python3 skills/scientific-computing-and-numerical-methods/inverse-problems-and-data-assimilation-starter/scripts/run_frontier_starter.py --out scratch/frontier/inverse-problems-and-data-assimilation-starter.json`.

Then inspect `refs.md` and `examples/resource_context.json` to promote the starter into a concrete executable workflow.
